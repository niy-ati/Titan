# TITAN Reports Archive

Curated verification, audit, architecture, and judge documentation moved here for GitHub organization.

## Note on regeneration

Verification scripts under `scripts/` may still **write new reports to the repository root** when executed (e.g. `npm run testnet:judge-verification`). Re-run those commands after moving files if you need fresh root-level outputs for CI.
