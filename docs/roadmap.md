# Roadmap

## Roadmap

The roadmap is gated by evidence.

Features move forward only when their upstream protocol, proof, and runtime assumptions are already classified.

### Near-term

* Mainnet MandateOS publish
* DeFi chain-verified reference proofs
* Allocation proof bundling
* Browser judge verification artifact

### Medium-term

* Oracle-gated allocation
* Policy objects
* Cross-protocol rebalancing
* Revenue routing expansion
* Treasury health scoring

### Long-term

* Autonomous treasury agent
* Treasury advisor
* Treasury credit score
* Dead-man treasury recovery
* Cross-chain programmable routing
