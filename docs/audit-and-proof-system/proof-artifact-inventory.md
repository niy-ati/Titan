# Proof artifact inventory

## Proof artifact inventory

This page documents the proof files explicitly referenced in the current TITAN docs set.

### Core proof files

* `deployment.json` — package deployment and upgrade record
* `judge-verification.json` — testnet judge workflow evidence
* `smart-wallet-rules-verification.json` — satellite rules publish, create, and execute evidence
* `defi-verification.json` — DeFi verification status and blockers
* `entrypoint-verification.json` — SDK entrypoint alignment checks
* `upgrade-result.json` — package upgrade result details

### What each file is for

These files establish deployment state, workflow evidence, proof verification inputs, and unresolved runtime gaps.
