# UI surface inventory

## UI surface inventory

This page documents the product surfaces that TITAN exposes or intentionally hides.

### Production workflow routes

* Treasury, objectives, and obligations
* Payroll, subscriptions, revenue, yield hub, guardian actions
* Smart wallet rules
* Allocation and protocol desks
* Portfolio, proof, audit, agents, templates, hub

### Hidden or unavailable routes

The product also contains simulation or dead-UI routes that are intentionally blocked from the production path.

### Why this matters

The route inventory is part of the truth model.

A review of TITAN is incomplete if it only documents chain-backed pages and ignores hidden or gated surfaces.
